## Vercel Deployment

- `vercel deploy` for production deployments
- Environment variables managed via Vercel dashboard or `vercel env`
- Preview deployments for every PR
- Edge Functions for low-latency API routes
- Serverless Functions for backend logic
- `vercel.json` for custom configuration (redirects, headers, rewrites)
- ISR (Incremental Static Regeneration) for dynamic content with static performance
